# Dataset

This project uses the ""Credit Card Fraud Detection"" dataset.

## Download

Download the dataset from Kaggle:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## File Required

After downloading, place the following file in your working directory:

```
creditcard.csv.zip
```

or extract it and use:

```
creditcard.csv
```

## Dataset Information

- Total Transactions: 284,807
- Fraud Cases: 492
- Normal Transactions: 284,315
- Features: 30
- Target Column: `Class`
